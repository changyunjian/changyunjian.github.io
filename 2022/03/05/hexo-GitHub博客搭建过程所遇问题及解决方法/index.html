Hexo+butterfly博客搭建

前期的博客搭建过程跟随 知乎教程，一路没遇到bug。直到主题美化和部署的时候遇到一些问题。记录如下：

butterfly主题图片更改

主题中的图片更改支持两种方式：

  1. 使用网上图片存储的地址。例如 https://images.pexels.com/photos/1072179/pexels-photo-1072179.jpeg。
  2. 使用本地存储的照片，先把照片放在 主题文件夹/source/img 下。

主题部署本地成功网上无主题

刚开始部署博客时，发现本地主题格式正常，网上则无主题样式。

![部署失败](屏幕截图 2022-03-06 135733.jpg “部署无主题”)
后面发现只需要改一下hexo文件夹下的配置文件_config.yml即可。
